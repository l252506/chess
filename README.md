# chess
THIS IS A SIMPLE CHESS GAME USING SFML
Features

Full chess rules:

Legal move validation

Check, checkmate, and stalemate detection

Castling

En passant

Pawn promotion

Drag-and-drop piece movement

Highlights:

Capturable enemy pieces in red

Last captured piece flashes red

King in check outlined in red

Status messages for side to move, check, and checkmate

Dependencies

SFML
 (Graphics, Window modules)

arial.ttf font for status messages

Piece textures in images/ folder:

wp.png, wr.png, wkn.png, wb.png, wq.png, wk.png

bp.png, br.png, bkn.png, bb.png, bq.png, bk.png
